La centrale inertielle

La centrale inertielle est une méthode de navigation utilisant des données géographiques rentrées au préalable ou bien en vol. Cette méthode est utilisée lorsque la météo et défavorable, que les points visuels sont trop espacés entre eux ou bien que la navigation sera longue et impécise.

Interface de la Centrale Inertielle

Composants du PCN :

Sélecteurs de Mode :

  • PREP : Préparation d’un waypoint.
  • INS : Navigation active.
  • BAD : Points d'intérêt balistiques ou de repère.
  • DEST : Sélection d’un waypoint actif.

Clavier Numérique : Pour entrer les coordonnées et autres paramètres.

Écran ALPHA : Affiche les informations des waypoints, positions, distances, etc.

Bouton VAL : Valider une entrée.

Sélecteur de Mode de l’INS : Contrôle l’état d’alignement et de navigation.

Alignement de la Centrale Inertielle

Étapes d’alignement :

  1. Allumer l’INS :
    • Tourner le sélecteur de mode INS sur ALN.
  2. Entrer la position initiale :
    • Basculer le PCN en mode PREP.
    • Taper le code "00" (position initiale).
    • Entrer les coordonnées de l’aérodrome à l'aide du clavier numérique (latitude, longitude et altitude).
    • Appuyer sur VAL pour confirmer.
  3. Attendre l’alignement complet :
    • L'écran ALPHA affiche le statut de l'alignement. Attendre qu'il passe en ALIGNÉ ou QUAL 9.
  4. Passer en mode NAV :
    • Tourner le sélecteur de mode INS sur NAV une fois l’alignement terminé.

⏳ Temps d’alignement : Environ 8 minutes pour un alignement complet. (Peut être réduit en alignement rapide, mais moins précis).

Navigation en Vol

  1. Sélectionner un waypoint actif
    • Basculer le PCN en mode DEST
    • Entrer le numéro du waypoint (ex. : 01 pour le premier waypoint) et valider avec VAL.
    • Les informations (cap, distance) apparaissent sur le HUD et l’écran ALPHA.
  2. Suivre les repères :
    • Le VTH affiche une flèche de guidage vers le waypoint ainsi que la distance.
    • L'IDN affiche une flèche de guidage vers le waypoint ainsi que la distance lorsqu'il est en mode NAV.

Gestion des Points BAD

Les points BAD (Balistiques ou Designation Airsol) sont des emplacements spécifiques utilisés pour l’attaque au sol.

Création d’un point BAD par coordonnées:

  1. Basculer en mode BAD sur le PCN.
  2. Entrer un numéro de point BAD (01, 02, etc.).
  3. Insérer les coordonnées du point et valider.

Création d’un point BAD par vecteur scalaire :

  1. Basculer en mode BAD sur le PCN.
  2. Entrer un relèvement dans la page θ.
  3. Insérer la distance dans la page ρ.

Ces points peuvent être utilisés pour le guidage des munitions ou comme repères pour des attaques air-sol.
