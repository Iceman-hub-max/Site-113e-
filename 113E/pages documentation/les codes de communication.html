Les codes de communications

Le Leader utilise des codes de communications étant des mots clés pour annoncer une action rapidement lors de phases critique de la mission. Voici les principaux :

  • Roulage = Indique le début de la phase de roulage.
  • Stop = Indique l'arrêt des aéronefs.
  • Alignement formation échalonnée = Indique que les avions s"alignent sur la piste en formation.
  • PC ON/OFF = Indique que les pilotes doivent mettre/couper la post combustion.
  • Top = Indique le début d'une action.
  • Rotation = Indique aux pilotes de décoller.
  • Train ON/OFF = Indique que les pilotes doivent étendre/renter le train d'atterrissage.
  • Volet ON/OFF = Indique que les pilotes doivent étendre/renter les volets.
  • Copy = Indique que l'information donnée est copiée.
  • Roger = Indique que l'information a été prit en compte.
  • Wilco = Indique que l'information sera prit en compte.
  • Affirm = Indique une réponse positive.
  • Négatif = Indique une réponse négative.
  • Stanbye = Indique le silence dans la fréquence.
  • Breack Breack = Indique de casser la formation.
  • Take over = Indique de reprendre la formation.
  • Breack UP/DOWN/RIGHT/LEFT = Indique de monter ou de décendre, de tournez à droite ou à gauche.
  • Fight on = Indique le début du combat.
  • Fox 1 = Indique le tir d'un missile du type FOX 1.
  • Fox 2 = Indique le tir d'un missile du type FOX 2.
  • Fox 3 = Indique le tir d'un missile du type FOX 3.
  • Bomb away = Indique le largage de bombes.
  • Rockets away = Indique le tir de rochettes.
  • Splash = Indique la confirmation que la cible est détruite.
  • Flaire = Indique que l'on se défent avec des contres mesures.
  • Visual = Indique que l'on a un visuel sur l'allié.
  • Blind = Indique que l'on n'a plus de visuel sur l'alliée.
  • Tally = Indique que l'on a un visuel sur la cible.
  • No joy = Indique que l'on n'a pas de visuel sur la cible.
  • Terminate = Indique que le combat est fini.
  • Request terminate = Indque que l'on demande la fin du combat.
  • No kit off = Indique un problème de sécurité.
  • Report back = Indique que les pilotes doivent rendre compte de leur situation.
